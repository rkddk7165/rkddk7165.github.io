PROBLEM

외부 통신이 트랜잭션을 길게 잡고 있었습니다

@Transactional 내부에 외부 Python 서버 통신이 포함되어 Blocking I/O 동안 DB 커넥션을 장기 점유했습니다. 주문 요청은 DB 저장을 빠르게 끝내고, 외부 서버 전송은 BlockingQueue 기반 워커 스레드로 분리했습니다.
