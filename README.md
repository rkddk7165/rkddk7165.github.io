# 강현민 포트폴리오

Java Spring 기반 백엔드 개발자 포트폴리오입니다.

## 구조

- `index.html`: 소개, 경험, 프로젝트, 연락처 원페이지
- `html/teenple.html`: TeenPle 상세 페이지
- `html/drone.html`: 배달의드론 상세 페이지
- `html/lms.html`: 학사정보시스템 상세 페이지
- `css/styles.css`: 공통 디자인 시스템
- `js/site.js`: 내비게이션, reveal, 카운터, 공통 footer
- `images/`: 프로젝트 이미지 자산
